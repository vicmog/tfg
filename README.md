# Small/Medium Business Management Project
**Negocio360**

This project aims to develop a **comprehensive tool for managing small businesses**, making it easier to handle inventory, sales, clients, and finances in a simple and efficient way.  

> ⚠️ **Note:** This is a **preliminary view** of the project. It is still under development, and many features are being implemented.

---

## Main Features (Planned)
- Product and inventory management  
- Sales invoicing  
- Customer and supplier management  
- Statistics  
- Intuitive admin dashboard  

---

## Technologies Used
- React Native  
- Node.js + Express  
- PostgreSQL  

*(Technology stack may be updated as the project progresses.)*

---

## Project Status
Currently in **early development**, which means:  
- 90% of features are incomplete.  
- There may be bugs or frequent changes in structure.  
- New improvements will be added continuously.
